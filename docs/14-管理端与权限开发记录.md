## 8. 默认补足项 / 当前假设

| 补足项 | 选择 | 原因 |
|--------|------|------|
| `AdminOnlyAttribute` 命名 | 类名 `AdminOnlyAttribute`，文件名为 `AdminOnlyFilter.cs` | C# 惯例：Attribute 类以 Attribute 结尾，使用时可省略后缀写 `[AdminOnly]`；文件名保持 Filter 便于理解 |
| MarkCompleted 不释放座位 | 保持 Seat.Status = Booked（1）不变 | 对齐 docs/08 §6.3 设计：管理员仅标记预约完成，不主动释放座位状态 |
| 管理端布局 vs 用户端 | _AdminLayout 深色导航 + container-fluid + 无页脚；Login 保持独立 | 对齐 docs/05 §4 管理端布局规范 |
| TempData 反馈 | 成功→alert-success，失败→alert-danger | 对齐 docs/05 §1.5 组件规范 |
| 删除座位确认弹窗 | 浏览器原生 `confirm()` | MVP 阶段避免引入额外依赖 |
| 新增/编辑使用 Modal | Bootstrap Modal 弹窗 | 对齐 docs/05 §5 组件清单 |
