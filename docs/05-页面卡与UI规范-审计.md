## C. 必改项（1 项）

### C1. 补充 P04 的"取消"按钮行为定义

在 P04 的状态表中增加一行：

| 状态 | 表现 |
|------|------|
| 用户点击"取消" | 取消返回上一页（P03 座位详情），不做任何数据修改 |

或者在信息层级中将"取消"改为"返回"，统一用语。

---

## D. 建议优化项（4 项）

1. **统一 P04 异常状态表述**：将状态表中的"数据异常"改为两段式说明：`(1) 座位 ID 无效 → 跳转 P02 并显示 "座位不存在"；(2) 其他异常 → 本页显示 alert-danger`。
2. **补充 P04 无 seatId 参数的处理**：增加一种状态：`无座位 ID → 显示"请先选择一个座位"并引导至 P02`。
3. **统一组件命名**：§5 组件清单中的 `.dropdown` 改为 `.form-select`，与 §1.5 保持一致。
4. **明确页面卡布局示意与真实线框图的关系**：在 §1.6 增加一句话说明"本节的 ASCII 布局示意即为 L1 线框图，原型工具绘制时以此为蓝本"。
