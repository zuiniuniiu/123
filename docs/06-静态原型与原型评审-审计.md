## D. 建议优化项（3 项）

1. **管理端导航改为 `<a>` 标签**：将 admin 页面顶部的 `<select>` 导航切换改为 `<ul class="navbar-nav">` + `<a class="nav-link">` 的链接触发方式，更符合 Bootstrap 导航规范，且静态页面下可直接点击跳转。
2. **统一用户端导航模式**：P04 底部补充与用户端其他页面一致的底部导航栏，或至少在 P04 添加"首页"快捷入口。
3. **P05/P02/P09 空状态 UI 预先在原型中体现**：在静态原型中加入一个隐藏的 `.d-none` 空状态模板区块，编码时可复用它。这不会破坏原型展示，但能为编码提供参考。
