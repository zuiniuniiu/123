## 5. @playwright/test 安装与 msedge 配置情况

### 安装命令

```bash
npm init -y
npm i -D @playwright/test
```

安装结果：Playwright 1.61.1 安装成功。

### 配置文件

`playwright.config.ts` 位于项目根目录：
- `baseURL`: http://localhost:5000
- `channel: 'msedge'` — 使用系统 Microsoft Edge
- `webServer` 配置了 `dotnet run` 命令 + `reuseExistingServer: true`
- 测试文件目录: `tests/`
