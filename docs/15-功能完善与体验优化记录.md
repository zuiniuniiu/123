## 7. 默认补足项 / 当前假设

| 补足项 | 选择 | 原因 |
|--------|------|------|
| CSRF 策略 | `[ValidateAntiForgeryToken]` + `@Html.AntiForgeryToken()` 标准模式 | ASP.NET Core 内置机制，无需额外库 |
| 状态徽标颜色 | docs/05 §1.5 为准：维护中=warning，已完成=info | 对齐页面卡 UI 规范 |
| ModelState 验证 | `asp-validation-summary="ModelOnly"` 显示顶层错误 | 简洁提示，优于逐字段校验 |
| 导航下拉回退 | 默认值"张三" | 系统设计为体验账号体系，无需严格认证 |
| 原型修复仅限 seat-list | 其他原型问题（R2/R4）标记为低优先级 | Sprint 2 审计确认为非阻塞 |
